# Day 11: 经典论文精读 — AlexNet 与 Transformer

## 第一部分：AlexNet

> 深度学习在计算机视觉领域的开山之作，首次在 ImageNet 上大幅超越传统方法。

### 一、图像预处理

- 最短边对齐 → 剪切图像中心位置（Center Crop）

### 二、五大关键设计（按重要性）

#### 1. ReLU 非线性激活

| | 饱和函数（Sigmoid/Tanh） | ReLU |
|------|------|------|
| 导数 | x 极大/小时导数 → 0 | 正区间恒为 1 |
| 梯度消失 | 严重 | 无 |
| 训练速度 | 慢 | **极快** |
| 归一化 | 需要输入归一化 | 不需要 |

> 只要有正输入，ReLU 就会发生学习。

#### 2. 多 GPU 训练

- 单 GPU 无法容纳过大网络
- 将核/神经元平分到 2 个 GPU
- GPU 之间直接通信（读写彼此内存）
- 只在某些层通信，**通过交叉验证选择**通信层
- 两个 GPU 学到不同特征（对颜色敏感性不同）

#### 3. 局部响应归一化（LRN）

#### 4. 重叠池化（Overlapping Pooling）

- 步长 < 窗口长度
- 不易过拟合
- 保留更多精度及信息

#### 5. 总体架构

```
5 个卷积层 + 3 个全连接层
     ↓
逐步提取特征（浅层→纹理，深层→语义）
     ↓
同时在 2 个 GPU 上搭建并训练
```

![AlexNet架构](https://i.postimg.cc/cJwybPzR/ping-mu-jie-tu-2026-07-13-162159.png)

### 三、减少过拟合

| 方法 | 说明 |
|------|------|
| **数据增强** | CPU 计算，无需占用磁盘空间 |
| **Dropout** | 迫使神经元无法持续依赖特定神经元 |

### 四、训练技巧

| 技巧 | 做法 |
|------|------|
| 权重初始化 | 均值 0、标准差 0.01 的高斯分布 |
| 偏置初始化 | 2/4/5 卷积层 + FC 隐藏层偏置 = 1（为 ReLU 提供正输入，加速早期学习）；其余 = 0 |
| 权重衰减 | 小权重衰减对学习很重要 |
| 学习率衰减 | 验证误差停止改善时，学习率 ÷ 10 |

### 五、关键结论

- 2 个 GPU 具有不同功能（对颜色敏感性不同）
- **深度至关重要**：移除任一卷积层都会导致性能下降

## 第二部分：Attention Is All You Need

> Transformer：完全基于注意力机制，摒弃循环结构，实现高度并行化。

### 一、动机

| RNN/LSTM 的问题 | Transformer 的解决 |
|------|------|
| 序列化计算无法并行 | 注意力机制全局并行 |
| 距离越远处理越复杂 | 自注意力 O(1) 路径长度 |
| 内存限制批处理 | 无循环状态，内存效率高 |

### 二、整体架构

![Transformer架构](https://i.postimg.cc/QN0LwQ68/ping-mu-jie-tu-2026-07-13-162118.png)

| 参数 | 值 |
|------|------|
| N（编码器/解码器层数） | 6 |
| d_model（隐层维度） | 512 |
| 注意力头数 | 8 |
| 每头维度 | 64（512 ÷ 8） |

> 残差连接要求所有子层输出维度 = 512
### 三、缩放点积注意力（Scaled Dot-Product Attention）
![image.png](https://i.postimg.cc/wMVscfHp/image.png)
$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

#### 计算流程

```
Q × K^T           → 计算相似度矩阵
÷ √d_k             → 缩放（防止梯度消失，dk越大点积越大）
+ Mask（解码器）    → 掩码（屏蔽未来位置）
→ Softmax          → 得到注意力权重
× V                → 权重加权求和
```

> 每个 token 的最终向量**包含所有其他 token 的含义**，并按注意力权重融合。

### 四、多头注意力（Multi-Head Attention）

#### 原理

| | 单头 | 多头 |
|------|------|------|
| QKV 维度 | 512×512 | 512×64（×8组） |
| 操作 | 直接计算 | 分组计算 → 拼接 → 线性层 |

#### 降维的意义

- 将特征聚合到相似语义子空间
- 不同头学到**不同表示**（语法/语义/位置等）

#### 线性层的意义

- 使聚合矩阵的特征**相互影响计算**
- 获得综合特征表示

### 五、前馈神经网络（FFN）

```
输入(512) → 升维(2048) → ReLU → 降维(512)
```

| 步骤 | 作用 |
|------|------|
| 升维 | 提取更精细的特征 |
| ReLU | 保留重要特征，过滤非正 |
| 降维 | 重新组合为综合特征 |

### 六、位置编码

- 使模型在**并行计算时仍能感知位置**
- 与 Embedding 相加后输入模型

### 七、编码器-解码器注意力

| | Q | K | V |
|------|------|------|------|
| 自注意力 | 当前层 | 当前层 | 当前层 |
| 交叉注意力 | 解码器 | 编码器输出 | 编码器输出 |

> 解码器的每个位置都能关注编码器的所有位置。

---

## 第三部分：要点总结

1. **AlexNet**：ReLU 替代 Sigmoid 解决了梯度消失，多 GPU 训练突破了算力瓶颈，深度至关重要
2. **Transformer**：自注意力使每个 token 都能直接关注所有位置，√d_k 缩放防止梯度消失
3. **多头注意力**：降维分组学习不同子空间特征，线性层融合综合表示
4. **FFN**：升维→非线性→降维，提取精细特征后重组
