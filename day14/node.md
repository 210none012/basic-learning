# Day 14: BERT 与 ResNet 论文精读

## 第一部分：BERT

> BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding

### 一、架构

多层**双向** Transformer 编码器，同时利用左右两侧上下文。

| | GPT | BERT |
|------|------|------|
| 方向 | 单向（左→右） | **双向** |
| 架构 | Decoder-only | Encoder-only |
| 适用 | 生成任务 | 理解任务 |

### 二、预训练

#### 1. Masked LM（掩码语言建模）

**目的**：双向预测比单向更有效。

**问题与方案**：

| 问题 | 方案 |
|------|------|
| 双向条件让单词间接看到自己 | 随机遮蔽 15% WordPiece token，仅预测被遮蔽词 |
| 预训练与微调不匹配（微调无 `[MASK]`） | 被选中的 15% token 中：80% → `[MASK]`、10% → 随机 token、10% → 不变 |

**优势**：编码器不知道要预测哪些词以及哪些词被替换，被迫保留每个 token 的**分布式上下文表示**。

**代价**：每批只预测 15%，收敛速度慢。

#### 2. NSP（下一句预测）

**目的**：训练模型理解句子间关系。

**方案**：
- 50%：B 是 A 的下一句（`IsNext`）
- 50%：B 是随机句子（`NotNext`）

#### 3. 训练配置

| 参数 | 值 |
|------|------|
| Optimizer | Adam（β₁=0.9, β₂=0.999） |
| Learning Rate | 1e−4 |
| L2 权重衰减 | 0.01 |
| 预热 | 前 10000 步线性预热 |
| Dropout | 所有层 0.1 |
| 激活函数 | GELU |
| 损失 | MLM + NSP 总和 |

#### 加速预训练技巧

> 先花大部分时间学短序列（128 token）掌握通用知识，再花少量时间学长序列（512 token）学习位置编码。

#### 4. 微调

| 参数 | 推荐范围 |
|------|------|
| Batch Size | 16, 32 |
| Learning Rate | 5e−5, 3e−5, 2e−5 |
| Epochs | 2, 3, 4 |

**实践建议**：
- 大数据集对超参数敏感性远低于小数据集
- 微调通常非常快，对这些参数穷举搜索是合理的
- 根据不同任务接入不同的输出层

#### 5. 消融实验

通过移除或修改某组件，证明原模型设计最优。

## 第二部分：ResNet

> Deep Residual Learning for Image Recognition

### 一、动机

**退化问题**：网络越深，训练误差反而升高——不是过拟合，是优化困难。

### 二、核心方法：残差学习

$$H(x) = F(x) + x$$

![残差块](https://i.postimg.cc/kghmYNrj/ping-mu-jie-tu-2026-07-16-111234.png)

通过 **Shortcut Connection** 执行恒等映射，将输入直接加到输出。

### 三、两种构建块

#### 维度相等时

$$y = F(x, \{W_i\}) + x$$

直接相加即可。

#### 维度不等时（下采样）

$$y = F(x, \{W_i\}) + W_s x$$

通过投影矩阵 $W_s$ 匹配维度。

### 四、优势

| 优势 | 说明 |
|------|------|
| 计算简单 | 只有加法，无额外参数 |
| 易于优化 | 不会像普通网络一样训练误差升高 |
| 深度灵活 | 允许训练更深的网络 |
| BN 正则化 | 依赖 Batch Normalization，省去 Dropout |
| 从高学习率开始 | BN 支持较高的初始学习率再衰减 |

### 五、关键发现

| 发现 | 说明 |
|------|------|
| 越深越显著 | 层数越深，残差效果越明显 |
| 浅层也受益 | 浅层网络中帮助**加快收敛** |
| 深层影响小 | 层数越深，残差网络对特征的**修改越小**（接近恒等映射） |

### 六、梯度视角

普通网络：梯度逐层连乘 → 梯度消失/爆炸

残差网络：
$$\frac{\partial H}{\partial x} = \frac{\partial F}{\partial x} + 1$$

> 常数 1 保证了梯度能直接回传，无论 $F$ 的梯度多小。

## 第三部分：要点总结

1. **BERT 双向性**：MLM 遮蔽 15% token（80/10/10 策略），NSP 理解句子关系
2. **BERT 微调**：LR 2e−5~5e−5，Epochs 2~4，参数搜索即可，训练极快
3. **ResNet 残差**：$H(x)=F(x)+x$，梯度 +1 防止消失
4. **ResNet 维度匹配**：等维度直接加，不等用 $W_s$ 投影
5. **BN 替代 Dropout**：ResNet 依赖 BN 正则化，省去 Dropout
